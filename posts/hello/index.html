整理项目结构

1.在工程目录下新建apps package

2.在工程目录下新建extra_apps package 存放需要修改的源码

3.新建目录media 存放图片等

4.新建目录templates 存放html,js等

5.新建目录db_tools 存放外部脚本

6.apps,extra_apps设置

7.setting.py设置apps，extra_apps加入Python跟搜索路径下

import sys
sys.path.insert(0, BASE_DIR)
sys.path.insert(0, os.path.join(BASE_DIR, 'apps'))
sys.path.insert(0, os.path.join(BASE_DIR, 'extra_apps'))

8.import 的时候系统的包放在最上面，第三方的包第二级别，自己的包第三级别

9.新建app

1.进去apps,执行django-admin startapp groups

2.修改setting.py，intstall_app 加上groups.apps.groupsconfig
